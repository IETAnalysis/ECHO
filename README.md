# ECHO

## ECHO: A Payload-Agnostic Cloud-Edge Hyperspherical Framework for Open-World Encrypted Traffic Classification


We understand that validating the model architecture and training process is crucial for peer review.
Therefore, we provide:
* Feature extraction code (feature-pre.py)
* Pretraining code (pretrain.py)
* Edge-side code (edge.py)
* Cloud training code (clould.py)

The specific hyperparameters have been indicated in the paper.



## Environment Setup
Python 3.8+ is recommended. Install the main dependencies using:
```bash
pip install torch torchvision numpy pandas scikit-learn scapy tqdm
```
